# 🚀 Cloud-Based Human Resource Management System (HRMS)

## 📌 Project Overview

This project is a **Cloud-Based Human Resource Management System (HRMS)** developed using Salesforce. It is designed to automate and manage core HR operations such as employee data management, leave processing, role-based access control, and reporting.

The system provides a centralized platform where HR personnel and employees can interact efficiently, reducing manual work and improving productivity.

---

## 🎯 Key Features

* 👤 Employee Management System
* 📝 Leave Management (Apply, Approve, Reject)
* 🔐 Role-Based Access Control (HR & Employee)
* 📊 Dashboards and Reports
* ⚙️ Automation using Flows & Approval Processes
* 🔔 Email Notifications

---

## 🧩 Modules Included

### 🔹 Employee Management

* Store and manage employee details
* Link employees with system users

### 🔹 Leave Management

* Employees can apply for leave
* HR can approve/reject requests
* Status tracking (Pending / Approved / Rejected)

### 🔹 Dashboard & Reports

* Total employees
* Leave request status
* Department-wise data

### 🔹 Security

* Profiles & Permission Sets
* Role Hierarchy
* Organization-Wide Defaults (OWD)
